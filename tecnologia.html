Realització de la pàgina web i automatització de l'hort

Per tot l'anterior he utilitzat:

  1. ArduinoBlocks per programar les ESP32.
  2. L'hort no té accés a Wifi ni 5G, pel que es comunica a distància amb una altra ESP mitjançant LoRa.
  3. Aquesta altra ESP32 està connectada a la WiFi i, en rebre les dades LoRa, les transmet a un full de càlcul, on es recullen tots els registres.
  4. Des del full de càlcul, s'envien les dades a aquesta pàgina per poder ser mostrades.

Per mostrar les dades de la pàgina Hort:

  • He utilitzat una placa ESP32 equipada amb sensors ambientals i actuadors per simular l'automatització d'un hort.
  • Visual Studio Code per poder realitzar els codis HTML, CSS i JavaScript per realitzar aquesta pàgina web.
  • Intel·ligencia artificial per generar un Script per a la fulla de càlcul de Google.
